``Custom Bias Generator`` - Description
===================================

``Custom Bias Generator`` is a Python software package that allows to build a synthetic dataset with a custom bias. 
The package is designed to be used in the context of machine learning and data science experiments, 
where the user needs to generate a synthetic dataset with a specific bias. 
The package provides a set of functionalities to generate a synthetic dataset and inject a bias into an existing dataset. 
The user can specify the desired bias by providing a probability mass function. 
The package is easy to use and can be integrated into existing machine learning pipelines.
